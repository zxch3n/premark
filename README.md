# @pretext-md

Fast Markdown layout and rendering built on top of `@chenglou/pretext`.

## Workspace

- `@pretext-md/parser`: streaming Markdown parser wrapper
- `@pretext-md/layout`: headless layout engine and streaming delta logic
- `@pretext-md/highlight`: Prism.js integration
- `@pretext-md/html-renderer`: pure HTML + CSS renderer
- `apps/playground`: interactive development playground
- `stories`: Storybook stories

## Commands

- `vp install`
- `vp run dev`
- `vp run build`
- `vp run test`
- `vp run typecheck`
- `vp run ready`
- `vp run storybook`
