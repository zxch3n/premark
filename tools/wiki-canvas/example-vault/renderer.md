# HTML Renderer

Pure HTML + CSS output. Absolute positioning, scoped `pmd-*` classes.

- Reads `DocumentLayout` from [[layout-engine]]
- Produces zero-JS markup
- Pairs with the canvas backend in [[wiki-canvas]]

\`\`\`html

<div class="pmd-doc">
  <div class="pmd-block pmd-block--paragraph"></div>
</div>
\`\`\`

Companion of [[premark]].
